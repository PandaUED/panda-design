# 通知栏

页面内栏状显示通知，可滚动循环播放，可定时或手动删除。

## API

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| props     | 说明                | - | - |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |
| style     | 自定义样式                             | object |    -    |
| onClick   | 点击按钮的点击回调函数                       |  func  |    -    |
