# 样式变量

包含颜色等全局样式变量

## API

| 属性       | 说明             | 类型     | 调用方法                                     |
| :------- | -------------- | ------ | ---------------------------------------- |
| color    | 颜色             | string | `style.color.type`               |
| gradient | 渐变色,支持自定义deg角度 | func   | `style.gradient.type(deg:number)` |
