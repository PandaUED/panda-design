# 头像

就是头像，可方可圆。大概也可以用于处理图片icon。

## API

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| size     | 头像大小                | number | 48 |
| shape     | 形状,圆形方形                | ['circle' / 'square'] | 'circle' |
| url     | 图片资源地址               | string | 无 |
| **OTHER** |                                   |        |         |
| className | 样式类名                              | string |    -    |
