# 输入框

说明用途

## API

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| size            | 样式                | string | 'small' |
| placeholder     | 占位                | string | - |
| left            | 左icon              | node   | - |
| right           | 有侧按钮             | node   | - |
| type            | 提供输入类型检测      | string | 'tel' or 'money' |
| value           | 外部可写入值，注意会导致组件刷新             | string   | - |
| **OTHER**       |                                      |        |         |
| classname       | 样式类名                              | string |    -    |
| onChange        | 输入框值变化的回调函数                  |  func  |    -    |
