# 卡片

用于组织信息和操作，通常也作为详细信息的入口。

## API

### Card

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |
| style     | 自定义样式                             | object |    -    |

### Card.Header

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| split | 是否显示下方分割线                              | bool |    false    |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |
| style     | 自定义样式                             | object |    -    |

### Card.Body

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| padding | 是否设置 `padding: 16px`                              | bool |    false    |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |
| style     | 自定义样式                             | object |    -    |

### Card.Footer

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| split | 是否显示上方分割线                              | bool |    false    |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |
| style     | 自定义样式                             | object |    -    |
