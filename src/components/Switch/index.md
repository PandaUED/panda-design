# 滑动开关

说明用途

## API

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| type      | 按钮类型               | string | 可选'square' |
| color     | 按钮激活的背景色        | string | '#4dd865' |
| active    | 按钮初始值             |  bool  |   false   |
| disable   | 是否禁用               |  bool  |   false   |
| **OTHER** |                                      |        |         |
| classname | 样式类名                              | string |    -    |
| onChange  | 按钮值改变时的回调函数                  |  func  |    -    |
