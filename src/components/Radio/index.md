# 单选按钮

单选

## API

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| value     | 单选框的值                | string | 'example' |
| group     | 单选框所在组，同一组内只能单选               | string | - |
| name     | 单选框的名称               | string | - |
| desc     | 单选框的详细介绍                | string | - |
| checked   | 按钮初始值             |  bool  |   false   |
| disabled   | 是否禁用               |  bool  |   false   |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |
| onClick   | 点击按钮的点击回调函数，对于单选框来说同change函数                       |  func  |    -    |
