# 列表

单个连续模块垂直排列，显示当前的内容、状态和可进行的操作

## API

### List


| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| classname | 样式类名                              | string |    -    |
| renderHeader     | 是否渲染列表头                      | object |    -    |
| renderFooter     | 是否渲染列表尾                      | object |    -    |


### List.Item

| 属性         | 说明                                |   类型   |   默认值   |
| :--------   | ---------------------------------   | :----: | :-----: |
| margin      | 是否向下分割                          | bool |    false    |
| onClick     | 点击时的处理                          | func |    -    |
