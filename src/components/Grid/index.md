# 栅格

24 栅格系统。

## API
### Row
| 属性       | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| gutter    | 栅格间间距                | number | 0 |
| align     | 栅格对齐方式               | string | 'top' |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |

### Col
| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| span      | 栅格占位格数                            | number | - |
| offset    | 栅格左侧的间隔格数，间隔内不可以有栅格      | number | 0 |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |
| height     | 自定义高度                            | number |    -    |
