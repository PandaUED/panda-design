# 复选框

多选输入

## API

| 属性        | 说明                                |   类型   |   默认值   |
| :-------- | --------------------------------- | :----: | :-----: |
| name     | 复选框的值                | string | 'example' |
| desc     | 复选框的详细介绍                | string | - |
| active    | 按钮初始值             |  bool  |   false   |
| disable   | 是否禁用               |  bool  |   false   |
| **OTHER** |                                   |        |         |
| classname | 样式类名                              | string |    -    |
| onChange   | 复选框值改变的回调函数                       |  func  |    -    |
