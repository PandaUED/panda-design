# 选项夹

用来展示产品信息

## API

| 属性              | 说明                     | 类型     | 默认值   |
| --------------- | ---------------------- | ------ | ----- |
| tabsData[Object]  | tab数据,元素为对象        | Array | -     |
| tabsData[].content| 选项夹的文字内容          | string | -     |
| tabsData[].icon   | ICON配置对象,具体按ICON组件文档             | Object | -     |
| tabsData[].title  | 模式为2-1时的标题         | string | -     |
| activeIndex       | 激活的tab索引             | number | 0     |
| **OTHER** |                                  |        |         |
