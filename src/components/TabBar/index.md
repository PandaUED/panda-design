# 选项夹

TabBar

## API

| 属性              | 说明                     | 类型     | 默认值   |
| ---------------   | ----------------------  | ------ | ----- |
| mode              | 选项夹展示的类型          | number | 0     |
| tabsData[Object]  | tab数据,请查看Tabs组件文档 | Array | -     |
| activeIndex       | 激活的tab索引             | number | 0     |
| **OTHER** |                                  |        |         |
